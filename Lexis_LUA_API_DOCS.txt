Lexis Lua is the Lua scripting API documented at docs.lexis.re. The site points developers to:

2. ENVIRONMENT
================================================================================
Source: https://docs.lexis.re/docs/getting-started/environment/

Purpose: Information about the environment available to a running script.

Permissions:
Scripts can be loaded with permissions. Some APIs may return empty state or do nothing when permission is missing.

Permission flags:
- natives: Calling natives
- global_read: Reading script globals
- global_write: Writing script globals
- local_read: Reading script locals
- local_write: Writing script locals
- memory_read: Reading game memory
- memory_write: Writing game memory
- https: Sending web requests
- file_read: Reading files
- file_write: Writing files
- stat_read: Reading stats
- stat_write: Writing stats
- transactions: Running transactions

JOAAT:
- joaat(text) can be used for game-native hash generation.
